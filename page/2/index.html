狭义相对论前传3•危机

发表于 2018-12-02 | 分类于 科学

先复习一下上一篇文章中麦克斯韦的三个发现:

  1. 他用四个方程概括了所有电磁现象；

  2. 他发现变化的电磁场可以互相产生，从而推导出电磁波；

  3. 他计算出电磁波的速度正好是光速，从而说明光其实就是电磁波。

一个细思恐极的问题

根据麦克斯韦的发现，光速本质上是电磁现象的结果，是可以用数学计算出来的。

不知道你注意到了没有，这里有个问题。

还记得第一篇文章里的思想实验吧。你站在速度是每小时 50 公里的游轮上，射出去一支速度是每小时 360 公里的箭，那这个“每小时 360 公里”就是相对于船上的你来说的。而相对于站在岸边的我来说，箭的速度就应该是每小时 50+360= 410 公里。这里面所谓的速度都是相对的。

那么，麦克斯韦方程组解出来的光速是相对于谁呢？

相对于光源

作为一个普通人，答案一般来说就是这样的。甚至还可以举出个例子:打开手电筒射出去一束光，这光速肯定是相对于手电筒的。是这样的吧？

物理学家告诉你，错。因为宇宙中有“双星系统”告诉了我们，光速跟光源的速度无关。

这可能需要你的一点想象力了。我们知道恒星会吸引周围的天体围绕它运动，那假如说现在有两颗恒星挨得很近，那么它们就会互相绕着对方旋转。而此时此刻，你作为一个旁观者，观察这个双星系统，一定是总有一颗恒星向着我们运动，另外一个恒星向着我们相反的方向运动。

而如果光速是相对于光源的速度，那么向我们走的这个恒星的光速就应该更快一点，离我们而去的恒星的光速应该更慢一点。

按照道理来说，因为双星离我们比较远，星光到达我们需要的时间很长。所以假如说速度有差异，那么我们足以观察到两颗星的星光有延迟。

可是天文学家观测了各种双星系统，从来都没有看到任何延迟。

这意味两个恒星的光速始终都是一样的！

这说明光速跟光源的速度无关。物理学家对此并不感到惊讶，因为他们早就意识到电磁波的速度和波源的运动速度无关，也就是光速和光源的运动速度无关。

这就意味着，无论你是朝平静的湖中扔下一颗石子，是垂直地从上空扔下去，还是斜着像打水漂一样地扔过去，这颗石子在水面上产生的涟都应该以相同的速度在水中扩散出去。

神奇吧？

相对于介质

物理学家相信，光既然是电磁波的一种波动，光速就一定是相对于某种“介质”的速度。

在此之前，你需要知道什么是“波动”。

波动

通俗地讲，你往平静的湖水里扔一块石头，水面上就会产生一层层的波纹，慢慢传播出去，这就是“波动”。请注意，在波往外传递的过程中，是波的形态在传播，湖水本身并没有往外传播。湖面上的水是被湖水局限在那里做来回运动而已。

你在海边看到海浪一层层地来到岸边，那些岸边的浪花只是岸边的水的波动，并不是远方的水跟着海浪一起来了。也就是在大尺度上，水并没有动，是波在相对于水而动。水是波传递的介质，波传递的仅仅是信息和能量，而不是物质。介质本身是不需要运动的。

如果你感到有点难以理解，你就想想声波。有个距离你十米远的人在说话，而你之所以能听到他的声音，是因为声波在空气中传递的结果。声波从那个人的嘴边到达了你的耳朵，但是那个人并没有把他嘴边的空气给吹到你这里。

既然水波是相对于水面的运动，声波是相对于空气的运动。那光作为电磁波，也是一种波动。它是不是也应该是相对于某种介质的运动呢？

这个假想中的介质，就被称为“以太”。

由此得到

  1. 光速与光源的速度无关。

  2. 波仅仅传递信息和能量，不会传递物质。

  3. 介质本身是不需要运动的。

狭义相对论前传2•一点电磁学内容

发表于 2018-12-02 | 分类于 科学

电和磁

如果我告诉你，你现在看到这行文字的电子设备就来自电磁学。你是不是对电磁学感觉亲近一点了呢？

你要知道:电磁学并不神秘。

  • 电就是电荷之间的相互作用。电子带负电，离子带正电。电子跟离子之间有一个吸引力，两个电子或者两个离子之间就有一个排斥力。这就是中学学的同性相斥，异性相吸。

  • 磁来源于电，电荷的运动会产生磁。一段导体中有电流，它周围就会有磁性。我们平时看到的磁铁，就是其中原子的排列很整齐，每个原子周围的电子的运动带来了磁力。

如果你想用物理学家的眼光理解电磁现象，那你还必须得掌握一个概念——场。

  • 两个电荷之间发生吸引，这个吸引力是怎么感觉到的呢？难道一个电荷隔空就能感到另一个电荷的存在吗？实际上，每个电荷都会在自己的周围形成一个电场。另一个电荷不是跟这个电荷直接发生相互作用，而是跟这个电荷的电场发生相互作用。确切地说，是所有的电场和磁场重叠在一起，形成一个总的电磁场，然后各个带电物质根据自己所在位置的电磁场来决定自己怎么运动。

最关键的是，电磁场可不是物理学家的想象，而是客观存在的东西——你完全可以用仪器探测出来。

场，就像是你坐的椅子一样真实。

光是电磁波的一种

1860 年代初期，伟大的麦克斯韦提出了一组方程，里面有四个方程，这四个方程描述了所有的电磁现象，这就是著名的麦克斯韦方程组。你去网上随便一搜索就可以找到公式和推导过程。我就不复制到这里了。

你只需要知道，麦克斯韦当时仅仅使用了数学来推论以下两个结论:

  • 变化的磁场能产生电场

  • 变化的电场又能产生磁场

电和磁在某种程度上是一回事。电场和磁场可以互相产生，就算没有电荷，用磁场也能产生电场。

诶！这个时候你想到，如果我用线圈弄一个震荡的电流，产生一个周期变化的磁场，那么这个周期变化磁场就能产生一个周期变化的电场，而这个周期变化的电场又能产生新的周期变化的磁场……岂不是说这个电磁场可以一直传播下去吗？

实际上这就是电磁波！二十多年以后人们真的在实验中制造了电磁波，给后世生活带来巨大的影响。不过，这是后话。

麦克斯韦现在可以用他的方程组直接计算这个电磁波的传播速度了。但是令他想不到的是，电磁波的速度，跟光速的数值，是一样的！

早在 1801 年人们就已经知道光是一种波，也是有速度的。但是人们并不知道光到底是怎么回事。

而现在麦克斯韦计算得出的电磁波的速度正好是光速，于是麦克斯韦大胆宣称，光，其实就是电磁波。后来人们证实果然是这样，我们平时所见的可见光无非就是特定频率的电磁波而已。

禅定时刻

物理学家再一次看破了红尘。人们之前知道，天上的东西和地上是一回事儿；匀速直线运动和静止是一回事；电和磁是一回事儿。

现在麦克斯韦说:光跟电磁场也是一回事儿。

狭义相对论前传1•建立坐标系

发表于 2018-12-02 | 分类于 科学

这篇文章是个开胃菜。用到的都是初中物理甚至是小学数学的知识。请放心阅读。

伽利略的相对论

假设你正在一艘豪华游轮上旅行。不过，虽然这艘游轮在海上开的速度很快，但是它却非常平稳，没有任何颠簸。游轮上有个全封闭的大厅，里面有游泳池，球场，餐馆……最重要的是，你可在里面做物理实验。

两个思想实验

现在，你在速度为每小时 50 公里的游轮上。以你自己建立一个坐标系，而我在地面上建立一个坐标系。那么你此时此刻相对于游轮是静止的，相对于我是运动的。

  • 设想一下，此时此刻你向前射出一支箭，箭相对于你的速度是每小时 360 公里，那么箭相对于我的速度就是每小时 360 + 50 = 410 公里。

  • 我们再来设想一下，你在游轮上的体育场里踢球，而我是守门员。此时此刻，你朝着游轮前进的方向射门，我只需要感到你射门的速度就可以拦下球了，不需要额外考虑游轮的速度。

这两个设想很有意思。你同样是在移动的游轮上，但是只要游轮的速度平稳不发生变化，你就无法判断它是在运动还是静止。

其实，我们生活的地球就是这样一个游轮。地球绕太阳公转的速度是每秒 29.8 公里，但因为地球走得几乎是一条直线，所以你完全感觉不到它正在高速前进。

不跳出自己的坐标系往外看，单凭做射箭、踢球这类的实验，你是无法区分你是待在运动还是静止的物体里面。匀速直线运动和静止没有本质区别，速度都是相对的。

这个道理，最早是物理学的祖师爷伽利略想明白的。这就叫“伽利略的相对论”。

而爱因斯坦的相对论，是对伽利略相对论的延伸。

总而言之，物理学家看破了运动。在物理学家的眼中，运动和静止其实是一回事儿。

不特殊论

我们中学学的牛顿运动定律里面，有两个定律都离不开伽利略的相对论。

第一定律说:在没有外力的作用的情况下，任何一个物体将会保持匀速直线运动或者是静止。这个定律告诉我们，匀速直线运动和静止的物体是一样的。它们都无需外力，也无需解释。

第二定律说:力会改变物体的运动方式。请注意，力不是运动的原因，因为即使没有力，物体本来也在匀速直线运动或者是静止。力是改变运动的原因。在一个理想的光滑平面上，一个滚动的乒乓球会一直前进。而生活中的乒乓球之所以会停下来，那是因为平面给它提供了摩擦力。一个物体要是一直在运动，那么并不需要解释。但是一个物体要是在运动中突然停下来了，才需要有个解释。

力只能带来加速度，单纯的速度跟力无关。匀速直线运动和静止都没有力，所以物理定律在游轮和地面是一样的。

讲到这里，你可能就明白了为什么单纯谈速度是没啥意义的。宇宙那么大，可能距离地球很遥远的一个星球，跟我们之间就有个特别高的相对速度。而那个星球上的物理定律和咱们这里的没有什么不一样。在那里的外星人看来他们是静止的，我们才是在高速运动。

所以相对论是物理学家的一个信念。这个信念也可以叫“不特殊论”：不管你的速度有多快，你的坐标系都不特殊。

禅定时刻

一般人小时候都做过公交车或者是火车，你会发现这篇文章讲的单纯看速度确实没意义。小时候的时候，在行驶的火车上看对面停止的火车，总有一种我在静止，它在快速后退的错觉。到底谁是动的，谁是快的，还真是说不清楚。

相对论专题的引言

发表于 2018-12-02 | 分类于 科学

我为什么要写这个专题

关于相对论，你肯定听过有关相对论的一些有趣的结论。比如一个物体在高速运动的时候，它的长度会变短，同时时间会变慢。

但是，相对论对绝大多数人来说是个神秘的理论。对于这么伟大的一个理论，如果对它的认知还停留在那些奇妙的结论上面，实在是有些可惜。

我是在初一的时候接触的宇宙学。当时看过一些关于相对论的著作，例如《时间简史》。

一个字:难！

霍金在写这本书的时候，是把它当做一本面向大众的科普书来写的。这本书里面没有涉及到晦涩难懂的理论和公式。但是，这对于普通人来说，还是有点难懂。

这时你可能会问，那我写这个专题是不是想要来科普相对论呢？答案是否定的。我这么点学识怎么敢称得上科普相对论。实际上，这个专题也并没有涉及到晦涩难懂的理论和公式——我只是来分享一些由相对论的结论和它们所引出的一些有趣的哲学问题——重点是在说相对论的思想。

你可能注意到了，相对论专题是独立在这座图书馆的科学分类之下的。原因很简单，相对论在世界上的地位实在是太重要了.相对论的质能方程E=mc²，甚至可以说得上是人类的智慧之巅。

同时，霍金在 2018 年 3 月 14 日去世了。说他是这个时代最伟大的物理学家也不为过。而且我认为他还有可能是前所未有的最伟大的物理学家。向他致敬。

没有理由不弄懂相对论

相对论本质上是关于时空的理论。正是因为有了相对论，我们才知道有黑洞这种东西，我们才知道空间居然会膨胀，我们才知道宇宙有个起源。

不得不承认，相对论是个出了名的难懂的理论。据说爱因斯坦刚刚发表狭义相对论的时候，全世界只有 2.5 个人能理解。等到相对论已经被物理学家广泛接受的时候，公众又理解不了。欧洲当时还出了一本书叫《一百个反对爱因斯坦的作家》 ，来说明相对论不靠谱。而爱因斯坦本人对此的回答是

“如果相对论真的错了，有一个人反对就够了。”

对，就是这么自信。

那相对论真的这么难吗？你要知道爱因斯坦发表狭义相对论是在 1905 年，距离今天已经 100 多年了。1905 年还是大清帝国了。网络上那些嘲笑脱离时代或者是过去一段时间的事情还拿出来说的人，经常说大清亡了之类的话。

但是，哪怕是你知道大清亡了，你还是不能理解一个清朝末年就已经出来的理论。

这个专题，我就想让你彻底讲明白相对论。

提前说明，相对论会对你的世界观产生重大影响。作为一个现代人，如果没理解爱因斯坦相对论，你就错过了这个世界最精彩的东西。

而且，你想想，如果相对论你都能琢磨清楚，那还有什么思想是高不可攀的呢？

我有一个好消息。相对论是一个简单并且干净利索的漂亮理论。

但简单不等于容易。简单的东西可以非常深刻。

到这里，你可能已经理解了相对论的美妙之处了。

让我们点击链接开始这趟理解相对论的旅行吧。

祝好运:）

感谢

本专题的直接诞生归功于《精英日课》专栏的万维钢老师。他在精英日课³的开篇就讲了相对论，这让我想起我初中对宇宙学的向往。接着这个机会，我翻出了存放于书柜已久的汪洁老师的著作《时间的形状》，开始着手于创作这个专题。

毕竟，我想在这座图书馆里面存放一些最伟大的思想。
